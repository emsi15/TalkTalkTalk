Do you want to keep it simple without thousands of commands and settings?
Well, then Chat Away is all you need: A simple chat application with some IRC like features.

Chat Away is a socket.io chat package including both server and client.

Read more about the server Read more about the client

The server

The chat server is built using Socket.io and running on a Node.js runtime environment.
Chat Away supports both multi user chatting and private messaging.

The server is easy to install and supports custom client implementations. Instructions can be found over at the Chat-Away GitHub repository.
